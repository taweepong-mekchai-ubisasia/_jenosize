# jenosize

## ขั้นตอนการติดตั้งสำหรับ backend 
ติดตั้ง package
```sh
npm i 
```
init Database 
```sh
npx prisma migrate dev --name init
```
```sh
node app.js
```

ข้อมูลการเชื่อมต่ออยู่ใน .env
